# YouTube_Channel_Analytics
A data analytics project analyzing YouTube channel performance using Python for data processing and Power BI for interactive dashboards. Includes data cleaning, visualization, and actionable insights on video engagement and trends.

Analyze YouTube channel data using Python and Power BI. 
Check video views, likes, comments, and category trends.

## Files
- Project_youtube_channel_analytics.ipynb → Python Notebook
- youtube_data_150.csv → Dataset
- Dashboard.pbix → Power BI dashboard
